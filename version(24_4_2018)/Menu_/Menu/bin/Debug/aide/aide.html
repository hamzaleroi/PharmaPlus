Les medicaments

  • Ajouter Un Medicaments
  • Rechercher Un Medicaments
  • Supprimer Un Medicaments




Ajouter Un Medicaments

Dans cette fenetre, l'utlisateur peut ajouter un medicaments au stock. D'abord ,il doit remplir le formulaire affiché,enuite il doit cliquer sur valider pour sauvegarder le médicament dans la base de données.

En cas d'erreur un message sera affiché indiquant le type d'erreur.







Rechercher Un Medicaments

L'utilisateur peut affecter une operation de recherche sur un médicament ,en introduisant son DCI ou bien D'autre informations (Marque,Dosage,Forme,Effet thérapeutique...).

Si le médicament existe dans le stock ,ses informations serons affichées,sinon son equivalent par rapport aux DCI ,et si ce dernier n'existe pas dans le stock ,un message indiquant qu'il n'existe pas dans le stock.






Supprimer Un Medicaments

Pour affecter une operation de suppression.D'abord ,l'utilaseur doit remplir le formulaire affiché,ensuite il doit cliquer sur valider pour valider la suppression ,un message sera affiché si l'operation a été effectuée avec succés.






Les produits Bien-étre

  • Ajouter Un Produit bien-étre
  • Rechercher Un Produit bien-étre
  • Supprimer Un Produit bien-étre




Ajouter Un Produit bien-étre

Dans cette fenetre, l'utlisateur peut ajouter un produit bien-étre au stock. D'abord ,il doit remplir le formulaire affiché,enuite il doit cliquer sur valider pour sauvegarder le produit dans la base de données.

En cas d'erreur un message sera affiché indiquant le type d'erreur.







Rechercher Un Produit bien-étre

L'utilisateur peut affecter une operation de recherche sur un produit bien-étre ,en introduisant la marque(Garnier, Avéne, L'oreal...) et le type (Shampooing, gel douche, créme, lait.... ).

Si le produit existe dans le stock ,ses informations serons affichées ,sinon un message indiquant qu'il n'existe pas dans le stock.






Supprimer Un Produit bien-étre

Pour affecter une operation de suppression.D'abord ,l'utilaseur doit remplir le formulaire affiché,ensuite il doit cliquer sur valider pour valider la suppression ,un message sera affiché si l'operation a été effectuée avec succés.






Saisir une facture

Pour saisir une facture, l'utilisateur doit remplir le nom du client et la date ainsi que le code du facture,ensuite il doit introduire les produits vendus. En cliquant sur Ajouter un produi,une petite fenetre sera affiché,l'utilisateur doit specifié le type de produit vendus (Médicament ou Produit Bien-étre) aprés il doit remplir le formulaire affiché,ensuite pour valider les informations il doit cliquer sur valider , s'il veut modifier ou supprimer resp un medicament aprés l'avoir valider ,il doit le selectionner dans la liste aprés appuyer sur le boutton Modifier un produit ou Supprimer un produit resp.

A la fin de cette operation le pharmacien valide la facture et il peut l'imprimer.





Saisir une commande

Cette commande est envoyée au fournisseur du produit (Medicament ou Bien-étre).
  • Commande d'un médicament
  • Commande d'un Bien-étre

Pour saisir une commande, l'utilisateur doit remplir le nom du fournisseur et la date ansi que le code du commande,ensuite il doit introduire les produits qu'il veut commander en remplissant un formulaire qui s'affiche quand on clique sur Ajouter un médicament ou Ajouter un produit.

A la fin de cette operation l'utilisateur valide la commande et l'envoie au fournisseur et il peut l'imprimer.





Commande d'autre pharmacies

Pour effectuer une commande à une autre pharmacie ,le pharmacien doit remplir le formulaire affiché et l'envoyer en cliquant le boutton Envoyer une commande. En cas d'erreur, un message éxplicatif sera affiché.





Les echanges

Cette fenetre a pour role d'informer l'utilisateur de tous les operations d'échanges éffectuées par lui.

l'utilisateur doit cliquer sur le boutton voir les informations pour voir tous les informations concernants les pharmacies qu'il a effectuer une operation d'échange avec (Nom de pharmacie,Numéro de téléphone,Fax,degré d'échange,l'adresse).
